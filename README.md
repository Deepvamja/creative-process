# Creative Process — React + Vite

Pixel‑perfect clone of the CodePen experience using React, GSAP, and Lenis.

## Scripts
- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Deploy (GitHub Pages)
1. Push this repo to GitHub.
2. Build: `npm run build` (outputs to `dist/`).
3. In GitHub repo: Settings → Pages → Deploy from branch → Choose `gh-pages` branch or `/docs` folder (alternatively use an action to publish `dist`).
